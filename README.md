# Partition iterators

Iterator adaptors that enumerate all partitions or all k-partitions of an iterator's content.

How to use with Cargo:

```toml
[dependencies]
partition-iterator = "0.1.0"
```

How to use in your crate:

```rust
use partition_iterator::PartitionIter;
```
